# BlinkLean Administrative Panel

This repository contains the administrative panel for BlinkLean, consisting of the React/Vite frontend and the Node.js/Express backend.

## Project Structure

*   **Frontend (`/`)**: React Single Page Application (SPA) built with Vite, Tailwind CSS, and React Router.
*   **Backend (`/server`)**: Node.js & Express API aggregation service, pre-configured for AWS Lambda serverless deployment via AWS SAM.

For instructions on how to deploy this stack to staging and production, please check the [AWS Production Integration and Deployment Guide (README-DEPLOYMENT.md)](file:///c:/Users/jeevithgowdasr/blinklean_admin-panel/README-DEPLOYMENT.md).
